# 転職ニュース自動収集プロジェクト

20代の転職に関する情報を毎日自動収集してダイジェストを生成するプロジェクト。
収集した情報をもとに、ソラ（@sora_tensyoku）としてThreads投稿案も自動生成する。

## ディレクトリ構成

- `digests/` - 毎日の転職ニュースダイジェスト
- `threads-drafts/` - Threads投稿案
- `weekly-ideas/` - 週次のnote記事ネタ出し
- `competitor-analysis/` - 競合Threadsアカウント分析
- `articles/` - 書いた記事を保存（手動）
- `.claude/rules/` - ルールファイル群

## 実行手順

1. CLAUDE.mdを読んでプロジェクト構成を確認する
2. .claude/rules/配下の全ファイルを読み込む
3. .last-check-state.mdで前回状態を確認する
4. 各ルールファイルに従って処理を実行する

## 発信者ペルソナ

- 名前：ソラ｜20代の転職攻略（@sora_tensyoku）
- 属性：転職2回経験、第二新卒からキャリアUP
- スタイル：攻略・ノウハウ7割、体験談3割
- スタンス：エージェントを敵視せず「賢く活用するツール」としてポジティブに紹介
- 禁止事項：暴露系・批判系の内容
- ターゲット：20代・初転職・第二新卒
- 一人称：使わない（客観的な攻略情報スタイル）
- 収益化：転職エージェントアフィリエイト + 有料note
