# 関心領域・フィルタリング設定

## 優先的に取り上げるトピック

### Tier 1（最優先 - 20代の転職に直結する実用情報）
- 書類選考・面接の通過率データ
- 職務経歴書・履歴書の書き方のコツ
- 転職エージェントの活用術・比較
- 20代・第二新卒向けの求人動向
- 年収アップの具体的な方法・データ
- 業界別の転職市場トレンド

### Tier 2（重要 - 転職活動全般のノウハウ）
- 面接でよく聞かれる質問と回答例
- 退職・入社のタイミングと手続き
- 転職活動のスケジュール管理
- 未経験転職・異業種転職のポイント
- リモートワーク・副業OKの求人動向

### Tier 3（補助 - 背景知識・トレンド）
- 有効求人倍率・完全失業率の推移
- 労働関連の法改正・制度変更
- 働き方改革・Z世代の仕事観
- AI・DXによる求人市場への影響

## 除外するトピック
- 40代以上向けのハイクラス転職
- 経営者・役員向けのヘッドハンティング
- 海外転職・外資系に特化した情報（軽く触れる程度はOK）
- フリーランス・独立に関する情報
- 暴露系（ブラック企業告発等）
- エージェント批判・ネガティブキャンペーン
- 政治的な労働政策の批評

## フィルタリングルール
- 「20代」「第二新卒」「初めての転職」に関連するものを優先する
- データや数字が含まれる情報を優先する（通過率○%、年収○万円など）
- エージェントに関する情報は「賢く活用する」視点でフィルタする（批判的な内容は除外）
- 同じトピックが連日続く場合は2日目以降はスキップする
